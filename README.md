## 🎨 Customization

### **Change Colors**
Edit the CSS color variables:
```css
:root {
    --primary: #0066ff;
    --secondary: #ff6b35;
    --accent: #00d4ff;
}
```

### **Add More Products**
Find the products array in JavaScript and add new items:
```javascript
{
    id: 9,
    name: 'Your Product',
    category: 'men',
    price: 1000,
    original: 1500,
    rating: 5,
    badge: 'New'
}
```

### **Update Prices**
All prices are in Pakistani Rupees. Edit directly in the product data.
